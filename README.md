## Ejecución local
*Compilación:*
Para compilar el ejecutable se emplea el siguiente codigo en terminal (Teniendo instalado MINGW)
```powershell
g++ main.cpp -o main -std=c++11
```

*Uso:*
```powershell
./main.exe 
```
